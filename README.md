# Проектирование осевого компрессора ГТЭУ

Инженерный Python-проект для расчёта и предварительного проектирования осевого компрессора стационарной газотурбинной установки.  
Репозиторий охватывает полный цикл: от подбора оптимальной комбинации безразмерных параметров до поступенчатого расчёта, профилирования лопаток и формирования итогового Excel-отчёта.

## Что сделано в проекте

- Реализована математическая модель компрессора с расчётом термогазодинамических параметров.
- Выполнен параметрический поиск допустимых решений по нескольким типам проточной части:
  - `D_к_const` (постоянный наружный диаметр),
  - `D_вт_const` (постоянный втулочный диаметр),
  - `D_ср_const` (постоянный средний диаметр).
- В поиск добавлены инженерные ограничения:
  - фильтрация по допустимой высоте лопаток и относительному диаметру втулки на выходе из компрессора,
  - отбор по эффективности (`η >= η_0`).
- Применена оптимизация вычислений:
  - схема поиска `coarse-to-fine`,
  - кэширование оценок кандидатов,
  - ускоренный расчёт `c_p` (с поддержкой `Numba`, если доступна).
- Автоматизирован экспорт и форматирование результатов в Excel.

## Исходные данные (базовый вариант)

| Параметр | Значение |
| :--- | :--- |
| Рабочее тело | Воздух |
| Массовый расход, `G_в` | 58.7 кг/с |
| Полная температура на входе, `Tа_полн` | 288 K |
| Полное давление на входе, `Pа_полн` | 1.013e5 Па |
| Требуемая степень повышения давления, `π_к_полн_0` | 18.2 |

## Логика расчётного пайплайна

1. `new_main_combi-V-4.ipynb`  
   Поиск и ранжирование допустимых комбинаций параметров, сохранение в `results/results_compressor_search.xlsx`.
2. `main.ipynb`  
   Детальный инженерный расчёт выбранного варианта:
   - начальные данные,
   - поступенчатый расчёт,
   - расчёт по высоте лопаток,
   - профилирование решёток/лопаток.
3. Формирование итогового отчёта `results/Итог_все_результаты.xlsx` и текстовых координат профилей в `Dots_results/`.

## Структура репозитория

- `main.ipynb` — основной расчёт и генерация итоговых артефактов.
- `new_main_combi-V-4.ipynb` — актуальный ноутбук параметрического поиска.
- `new_main_combi-V-3.ipynb` — предыдущая версия поиска (история развития модели).
- `resources/` — входные таблицы и методические материалы.
- `results/` — автоматически сформированные Excel-результаты и изображения.
- `Dots_results/` — координаты профилей лопаток по сечениям.
- `Blueprints/` — чертежные материалы.

## Как запустить

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
jupyter notebook
```

Рекомендуемая последовательность:
1. Выполнить `new_main_combi-V-4.ipynb` для подбора варианта.
2. Подставить выбранные параметры в `main.ipynb`.
3. Выполнить `main.ipynb` для получения полного комплекта результатов.

- `results/Итог_все_результаты.xlsx` — сводный итоговый файл.
