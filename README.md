# Compressor_Kurs

# Проектирование осевого компрессора стационарной ГТЭУ

Данный репозиторий содержит материалы курсового проекта по расчету и проектированию осевого компрессора для стационарной газотурбинной энергетической установки [2].

## Исходные данные для проектирования
Согласно заданию, расчет производится по следующим параметрам [2]:

| Параметр | Значение |
| :--- | :--- |
| Рабочее тело | Воздух |
| Массовый расход ($G_{вх}$) | 58,7 кг/сек |
| Температура на входе ($T_{вх}^*$) | 288 К |
| Давление на входе ($P_{вх}^*$) | 101300 Па |
| Степень повышения давления ($\Pi_k^*$) | 18,2 |
| Частота вращения вала ($n$) | 8600 об/мин |

## Состав репозитория
*   main.ipynb — основной Jupyter Notebook с реализацией расчетных алгоритмов [1].
*   расчет_ступеней.xlsx — таблица итоговыми данными пошагового расчета ступеней компрессора [1].
